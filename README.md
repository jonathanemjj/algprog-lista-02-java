## 📌 Descrição

Este projeto contém a implementação de 5 exercícios básicos em Java, com o objetivo de praticar lógica de programação, entrada de dados e operações matemáticas utilizando a linguagem Java.

Os programas foram desenvolvidos utilizando o **Visual Studio Code**, mas podem ser executados em qualquer IDE compatível com Java.

---

## 🧠 Conteúdos abordados

* Entrada de dados com `Scanner`
* Operações matemáticas básicas
* Estrutura sequencial
* Uso de variáveis
* Saída de dados no console

## 🚀 Exercícios

### 🔢 Exercício 1

Solicita um número ao usuário e exibe a mensagem:

```
O número informado foi [número]
```

---

### ➕ Exercício 2

Recebe dois números e realiza:

* Soma
* Subtração
* Multiplicação

---

### 📊 Exercício 3

Recebe 4 notas bimestrais e calcula a média:

```
Média = (n1 + n2 + n3 + n4) / 4
```

---

### ⛽ Exercício 4

Calcula o valor total de combustível:

```
Total = preço do litro × quantidade de litros
```

---

### ✈️ Exercício 5 (Eurotrip)

Calcula o custo total de uma viagem para:

* Alemanha
* Portugal
* Itália

Levando em consideração o número de pessoas:

```
Total = (Alemanha + Portugal + Itália) × pessoas
```
